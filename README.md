# 플러터 앱 [ tabe ] 

일본 최대 쿠폰앱 홋또페퍼 API를 이용한 일본현지 맛집앱!  
앱은 일본어로 되어있습니다!  
일본 현지 맛집을 바로 알 수 있어요.　🍣🍺

# 기술

- HTTP
- flutter_dotenv
- ListView
- SingleChildScrollView
